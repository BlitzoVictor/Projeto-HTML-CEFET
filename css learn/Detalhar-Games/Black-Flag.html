Assassin's Creed IV: Black Flag

Assassin's Creed IV: Black Flag é um jogo de ação e aventura ambientado no século XVIII, onde o jogador controla Edward Kenway, um pirata que se envolve na luta entre Assassinos e Templários. Edward, inicialmente em busca de riqueza, acaba se tornando um Assassino e se vê em meio a intrigas envolvendo o artefato "Fruto do Éden"

Requisitos do Sistema

Requisitos Mínimos

  • Sistema Operacional: Windows Vista SP2 / 7 / 8
  • Processador: Dual-core CPU 2.4 GHz
  • Memória: 2 GB de RAM
  • Placa de Vídeo: NVIDIA GeForce 460 / ATI Radeon HD 6850 com 768 MB
  • DirectX: Versão 11
  • Armazenamento: 30 GB de espaço disponível

Requisitos Recomendados

  • Sistema Operacional: Windows 7 / 8
  • Processador: Quad-core CPU 3.0 GHz
  • Memória: 4 GB de RAM
  • Placa de Vídeo: NVIDIA GeForce GTX 560 / ATI Radeon HD 7770 com 1 GB
  • DirectX: Versão 11
  • Armazenamento: 30 GB de espaço disponível
